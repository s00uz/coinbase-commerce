3 Understanding Credit Types

Different services in CDP use specific credit systems with distinct billing mechanisms:

‚ö° Paymaster Credits

  • Purpose: Sponsor gas fees for user operations on Base network
  • Billing: Charged based on actual gas consumption by sponsored transactions
  • Limits: Service interruption for unpaid amounts exceeding $10,000
  • Usage: Ideal for applications wanting to abstract gas costs from end users

üåê RPC API Credits

  • Free Tier: 7,500 API calls per 5-second window
  • Rate Limiting: Automatic throttling when limits are exceeded
  • Premium Plans: Higher rate limits available for paid tiers
  • Usage Tracking: Real-time monitoring of API consumption
